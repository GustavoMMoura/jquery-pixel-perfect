jQuery-Pixel-Perfect

A simple and lightweight tool for convenient work with design mockups in browser

Download (v1.0) Docs

What is it?

jQuery-Pixel-Perfect is a small plugin that allows a convenient way to impose design layout over the developed site. So you have no lines and measurements can write CSS code that pixel-matching design layout. Try it ride now, press [S] key

How to install?

  1. Download the latest version of jQuery and jQuery-Pixel-Perfect css and js files
  2. Add files to your project <link href="jquery-pixel-perfect.css" rel="stylesheet">
    <script src="https://code.jquery.com/jquery.js" type="text/javascript"></script>
    <script src="jquery-pixel-perfect.js" type="text/javascript"></script>
  3. Create overlay image for page you are coding and put it in some folder in your project. For example in ../images/mockup.png
  4. Add initial js code which bind "body" tag with jQuery-Pixel-Perfect method: $(document).ready(function(){
      $('body').pixelPerfect();
    })

How to use?

  1. Use the following buttons to:
    [Up] / [Down] / [Left] / [Right] keys or mouse for moving mockup
    [Q] / [W] keys for changing opacity value
    [S] key to show/hide mockup
    [F] key to change "absolute/fixed" positioning property of mockup
  2. Actually you can change default parameters like an actions of keyboard keys, image path, mockup draggable statement. For this you need to specify the input parameters of the jQuery-Pixel-Perfect method. $(document).ready(function(){
      $('body').pixelPerfect({
        path: "images/mockup.png", //default mockup path
        draggable: true, //draggable state
        topBtnCode: 38, //default is "Up" key
        rightBtnCode: 39, //default is "Right" key
        bottomBtnCode: 40, //default is "Down" key
        leftBtnCode: 37, //default is "Left" key
        opacityIncBtn: 81, //default is "Q" key
        opacityDecBtn: 87, //defauls is "W" key
        changePositionBtn: 70, //default is "F" key
        visibility: 83 //default is "S" key
     });
    })
